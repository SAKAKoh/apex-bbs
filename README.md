# Apexフレンズ掲示板

### 概要
Apex Legendsのプレイヤー同士が仲間募集を行える掲示板アプリを開発しました。

ゲーム内で自分に合うプレイヤーを見つけにくいという課題を感じ、
条件を絞り込んで募集できる仕組みを作成しました。

---

### 機能
- 募集投稿機能（フォーム入力 → データベース保存）
- 投稿一覧表示機能
- 条件選択（ゲームモード・ランク帯・機種・ボイスチャットなど）

---

### 使用技術
- HTML / CSS
- PHP
- MySQL

---

### 工夫した点
- ユーザーが自分に合うプレイヤーを見つけやすいように、
  ランク帯やボイスチャットの有無など複数条件を指定できる設計にしました。
- htmlspecialchars を使用し、
  セキュリティ面も対策しました。

---

### 学んだこと
- ユーザー目線で機能を考えることの大切さを実感しました。
- エラー発生時には、原因の切り分けを行いながら修正と検証を繰り返し、解決しました。
- 解決できない場合は代替の実装方法を調べて対応しました。
